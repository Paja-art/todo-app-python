Tento projekt je jednoduchá konzolová To‑Do aplikace napsaná v jazyce Python.
Demonstruje objektově orientovaný přístup a využití Command design patternu, který odděluje logiku příkazů od hlavního programu.

Aplikace umožňuje:

přidávat úkoly

označovat úkoly jako dokončené

mazat úkoly

zobrazovat seznam úkolů

Projekt je určen jako semestrální práce do předmětu zaměřeného na OOP.
